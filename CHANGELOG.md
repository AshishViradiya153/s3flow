# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-04-18

### Added

- Initial public release: streaming S3 prefix → ZIP / tar / tar.gz, checkpoints, prepared NDJSON index, optional multipart upload (`s3flow/platform`), BullMQ helpers (`s3flow/bullmq`), GCS and Azure Blob storage providers, CLI, metrics, root README, and a **`docs/`** handbook (guides, troubleshooting, export reference) shipped in the npm package.
